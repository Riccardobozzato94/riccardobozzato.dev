🔍 Phase 1: Process Map 60 min

Map each process end-to-end (trigger → outcome). Mark every handoff, approval gate, and tool change. Identify: handoff density (how many times work changes hands), tool switches (how many tools touch one process), approval bottlenecks (where work waits for a person).

📊 Phase 2: Metric Baseline 60 min

For each process, measure Lead Time (trigger to outcome), Active Time (actual work), and Wait Time (queues). Efficiency = Active ÷ Total. Below 40%? Your team spends more time waiting than working.

Process Lead Time Active Time Wait Time Efficiency Data Source

Efficiency = Active Time ÷ (Active Time + Wait Time) × 100. If you can't get this data in 10 minutes, your process is invisible.

🎯 Phase 3: Priority Matrix 60 min

Score each process on two axes: Operational Impact (how much does this affect revenue/delivery? 1-5) and Fixability (how much effort to improve? 1-5 where 1 = trivial, 5 = massive). Plot below.
